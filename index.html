Recipe Record

A record of my personal journey throughout the culinary world, as well as a first personal project for web development. Expect either a dead page that never gets hosted or something that may actually be turned into a self-updating website! Plans include actual pages, as well as the ability for the website to dynamically display “cards”, or blocks of text for each time I actually make a recipe. Actual website visual design is also in the works.

If I were to list out the features I plan to potentially add, then:

  • Multiple pages
  • Sidebar or top menu bar for navigation
  • A complete mockup for a "base" recipe page
  • Database integration
  • Ability for users to add their own creations/recipes/modifications/feedback as cards, including details such as difficulty, time taken, pictures, substitutes, etc.
  • Ability for users to rate others' cards
  • Some algorithm to sort based on ratings
  • Tags system to organize user cards
  • Algorithm to detect or recommend relevant tags to a user's card
  • Link recipes from other sources as their own cards

If you're curious about what tutorial I'm using, I'm simply following along on the MDN web docs beginner tutorial
